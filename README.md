## 🚀 Quick Start

### Prerequisites

- Hetzner CX31 instance (2 vCPU, 8GB RAM) or higher
- Docker & Docker Compose
- Kubernetes (K3s)
- CUDA-capable GPU (optional, for Triton acceleration)
- Git

### Deployment

```bash
# Clone the repository
git clone https://github.com/brian95240/autonomous-vertex-ken-system.git
cd autonomous-vertex-ken-system

# Set up environment variables
cp .env.example .env
# Edit .env with your configurations

# Deploy infrastructure
./automation/scripts/deploy.sh

# Verify deployment
./automation/scripts/test.sh
```
